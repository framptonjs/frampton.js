Class: Cache

Frampton. Cache

new Cache(options)

Simple cache that removes items based on least recently used (LRU).
Parameters:
Name Type Description
options Object A hash of options to configure the cache. Currently only supports LIMIT (the max number of items in cache) and TIMEOUT (how long an entry should be valid).
