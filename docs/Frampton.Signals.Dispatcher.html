Class: Dispatcher

Frampton.Signals. Dispatcher

(private) new Dispatcher(stream)

Dispatcher is a helper object that helps the a stream manage its Outlets. A new instance of the Dispatcher is created for each new stream. The owning stream inherits references to its dispatcher's subscribe, broadcast and clear methods.
